# LibraryHub

LibraryHub, ASP.NET Core MVC ile geliştirilmiş basit bir kütüphane yönetim sistemidir.

## Özellikler

- Admin girişi
- Kitap, yazar, kategori ve üye yönetimi
- Kitap ödünç verme ve iade alma
- Dashboard istatistikleri
- Arama ve filtreleme ekranları
- SQLite ve Entity Framework Core kullanımı

## Kullanılan Teknolojiler

- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- Bootstrap

## Çalıştırma

```bash
dotnet restore
dotnet build
dotnet run
```


İlk çalıştırmada veritabanı otomatik oluşturulur. Varsayılan admin bilgileri:

- Kullanıcı adı: `admin`
- Şifre: `admin123`

  Projeyi çalıştırmak için .NET 10 SDK gereklidir.
